🌱 Welcome to FarmConnect 👋
This is an Expo project bootstrapped with create-expo-app.

📱 FarmConnect Overview
FarmConnect is a mobile app designed to empower farmers by providing tools to:

-Manage farm tasks

-Create marketplace listings

-Connect buyers and sellers

🛠 Tech Stack
-React Native

-Expo

-Firebase Authentication

-Firestore

-Cloudinary

✨ Features
-User authentication

-Planner system

-Marketplace listings

-Persistent data storage

🚀 Getting Started
1. Install dependencies

npm install

2. Start the app
   
npx expo start

You’ll see options to run the app in:

-Development build

-Android emulator

-iOS simulator

Expo Go (sandbox environment)

🧑‍💻 Development Workflow
Edit files inside the app directory (file-based routing).

Reset the project when ready:


npm run reset-project

This moves starter code to app-example and creates a fresh app directory.

⚙️ Additional Setup
-ESLint & Prettier → Run npx expo lint or follow the guide.

-Unit Testing with Jest → Follow Expo’s testing guide.

-TypeScript setup → Learn how TypeScript integrates with Expo.
